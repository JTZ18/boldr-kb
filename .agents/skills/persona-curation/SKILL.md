# Persona Curation

Use this skill for persona entries under `kb/personas/`.

## Steps

1. Review existing persona entries for the same axis or signal cluster.
2. Update an existing persona when the new evidence is refinement.
3. Create a new persona only when the signals are meaningfully distinct.
4. Keep the entry factual and evidence-linked.
5. Render and validate the index.

## Rules

- Use `domain: persona`.
- Put the persona axis in `themes`.
- Add concrete trigger keywords in the body.
- Avoid demographic claims unless the source explicitly supports them.
