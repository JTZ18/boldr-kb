# Sentiment Briefing

Use this skill for `external-intel/` and brief-ready findings.

## Steps

1. Read provided external-intel artifacts before summarizing.
2. Separate customer KB facts from external sentiment.
3. Write dated findings under the requested path.
4. Mark source URLs or source artifact paths.
5. Run validation.

## Rules

- Do not turn market chatter into product or policy facts.
- Flag weak signals as weak.
- Keep recommendations traceable to source artifacts.
